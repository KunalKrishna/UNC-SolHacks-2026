## How to Qualify

### Step 1: Fork This Repository

1. Click the **Fork** button at the top-right of this repo
2. This creates your own copy at `github.com/YOUR_USERNAME/UNC-SolHacks-2026`
3. Clone your fork locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/UNC-SolHacks-2026.git
   cd UNC-SolHacks-2026
   ```

### Step 2: Set Up Your Environment

Follow the [QUICKSTART.md](QUICKSTART.md) guide to install Python, VS Code,
and GitHub Copilot.

### Step 3: Complete at Least 3 Exercises

Work through the exercises in the `examples/` folder. **You must complete a
minimum of 3 exercises** to qualify for a voucher.

| # | Exercise | What You'll Do |
|---|----------|---------------|
| 1 | [Autocomplete](examples/01_autocomplete/) | Let Copilot finish incomplete functions |
| 2 | [Writing Tests](examples/02_writing_tests/) | Generate tests for a Calculator class |
| 3 | [Refactoring](examples/03_refactoring/) | Clean up messy code with Copilot |
| 4 | [Documentation](examples/04_documentation/) | Auto-generate docstrings |
| 5 | [Debugging](examples/05_debugging/) | Find and fix 5 intentional bugs |
| 6 | [Build an API](examples/06_build_api/) | Build a REST API with Copilot |

**Bonus:** Build something original in the [Starter Project](starter_project/)
folder for extra credit!

### Step 4: Push Your Work

Commit and push your completed exercises to **your fork**:

```bash
git add .
git commit -m "Completed Copilot Learning Lab exercises"
git push origin main
```

> **Important:** Your forked repository must be **public** so organizers can
> verify your work.

### Step 5: Submit Your Entry

Open an issue using our submission form:

👉 **[Submit for Voucher Giveaway](https://github.com/esola-thomas/UNC-SolHacks-2026/issues/new?template=voucher-submission.yaml)** 👈

Fill out all required fields including:
- Your name and UNC email
- Your GitHub username
- Link to your forked repository
- Which exercises you completed
- Your preferred certification exam

## Rules & FAQ

### Rules

1. **UNC students only** — you must have a valid `@unc.edu` email
2. **One voucher per student** — duplicate submissions will be disqualified
3. **Minimum 3 exercises** — complete at least 3 of the 6 exercises
4. **Public fork required** — we need to verify your work
5. **First come, first served** — submit early, vouchers are limited
6. **Your own work** — you must complete the exercises yourself using Copilot
   (copying someone else's fork doesn't count)

### How Will Vouchers Be Distributed?

- Submissions are reviewed in the order received (by issue timestamp)
- Organizers will verify your fork contains completed exercises
- Approved students receive their voucher code via the email provided
- You'll have **6 months** to schedule and take the exam

### What Counts as "Completed"?

- **Autocomplete:** All 8 functions implemented and self-checks pass
- **Writing Tests:** At least 10 test assertions written
- **Refactoring:** Code refactored and self-checks still pass
- **Documentation:** Docstrings added to all functions
- **Debugging:** All 5 bugs found and fixed
- **Build an API:** At least 3 endpoints implemented
- **Starter Project:** Any original code beyond the template

### Can I Use Copilot Chat?

**Yes!** In fact, we encourage it. The whole point is to learn AI-assisted
development. Use inline suggestions, Copilot Chat, or both.

### What If Vouchers Run Out?

We'll close the submission form once all vouchers are claimed and update this
page. Follow the repo to get notified.

### I Don't Have GitHub Copilot — Can I Still Participate?

GitHub Copilot is **free for students** through
[GitHub Education](https://education.github.com/). Sign up with your `.edu`
email to get access.
